import sys
import subprocess

def ensure(pkg_name, module_name=None):
    try:
        __import__(module_name or pkg_name)
    except ModuleNotFoundError:
        print(f"'{pkg_name}' not found – installing via pip...")
        subprocess.check_call([sys.executable, "-m", "pip", "install", pkg_name])

for pkg, mod in [
    ("flask", None),
    ("pandas", None),
    ("plotly", None),
    ("pyserial", "serial"),
]:
    ensure(pkg, mod)

Project:

CC Control Code (Web Edition)

Description:

  • Python Flask web app for controlling a continuous casting system via serial (G-code).
  • Includes real-time temperature plotting and experiment state-machine logic.
  • Auto-installs required Python packages at runtime.
  • Interactive 3D plot with Plotly, live charting with Chart.js, and CSV result export.

Usage:

  1. Connect the hardware (set correct COM port).
  2. Run python CC\ Control\ Code\ (Web\ Edition).py
  3. Follow prompts and control/monitor from your browser.
  4. See repo for detailed setup instructions and full code.

Features:

  • Live temperature and process graphing
  • Interactive plotting of experimental results
  • Automated G-code control and override
  • Web-based data entry and export
